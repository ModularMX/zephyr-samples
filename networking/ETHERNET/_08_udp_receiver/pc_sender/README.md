# PC UDP Sender for Zephyr UDP Receiver Example

This script provides a simple UDP sender to test the Zephyr UDP receiver (Example 8).

## Usage

1. Make sure your Zephyr board is running Example 8 (UDP Receiver).
2. Run this script on your PC:

```bash
python udp_sender_test.py
```

The sender will transmit UDP packets to the Zephyr board for testing.
