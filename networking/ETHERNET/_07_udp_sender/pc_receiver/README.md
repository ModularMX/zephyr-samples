# PC UDP Receiver for Zephyr UDP Sender Example

This script provides a simple UDP receiver to test the Zephyr UDP sender (Example 7).

## Usage

1. Make sure your Zephyr board is running Example 7 (UDP Sender).
2. Run this script on your PC:

```bash
python udp_receiver_test.py
```

The receiver will display incoming UDP packets from the Zephyr board.
